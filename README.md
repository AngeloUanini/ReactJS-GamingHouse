# GamingHouse - Tienda E-commerce de Productos Gaming

**GamingHouse** es una aplicación web desarrollada con **React.js** que simula una tienda online de artículos de computación orientados al gaming.  
El proyecto forma parte de la entrega final del curso de **React JS - Coderhouse**.

---

## Características principales

- Listado dinámico de productos obtenido desde **Firebase Firestore**  
- Detalle individual de cada producto  
- Carrito de compras con manejo global mediante **Context API**  
- Formulario de checkout con envío de la orden a Firestore  
- Navegación completa con **React Router DOM**  
- Renderizado condicional para mensajes, loaders y vacíos de contenido  
- Estilos personalizados con **CSS puro**

---

## Tecnologías utilizadas

| Tecnología | Uso principal |
|-------------|----------------|
| **React.js (Vite)** | Framework base de la aplicación |
| **React Router DOM** | Navegación SPA entre vistas |
| **Firebase / Firestore** | Base de datos y almacenamiento de órdenes |
| **Context API** | Gestión global del carrito de compras |
| **CSS Modules** | Estilos personalizados |

## Instalación y ejecución local

1. Clonar el repositorio:

2. Instalar dependecias:
